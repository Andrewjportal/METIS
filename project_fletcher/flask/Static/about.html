YouTopics

About YouTopics

Overview

Given a user-generated list of keywords representing a topic,
YouTopics measures the similarity between the user's topic
and a youtube video's topic. It achieves this by extracting keywords
from the video's transcript or autogenerated captions and
comparing the video's keywords with the user generated ones.

Use

YouTopics allows the user to quickly analyze the relevance of a video.
YouTube is one the world's most popular websites. YouTopics is a first
step at accessing this vast source of information.

Method

A TextRank algorithm is used to extract keywords and summary sentences
from the video's transcript or autogenerated captions.The keywords are
converted to word vectors from pre-trained GloVe word
embeddings. The word vectors are summed and then compared to a vector
derived from the user's keywords using cosine similarity.

Future Work

Add a database storage function for user defined topics. Then apply
database of topics for more robust topic analysis. Add multi-topic
